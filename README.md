# Inventario-RIWI
# Sistema de Inventario en Python

## Descripción

Este programa permite registrar un producto en un inventario mediante la consola.
El usuario ingresa el nombre del producto, su precio y la cantidad disponible.
El sistema valida los datos ingresados, calcula el costo total del producto y muestra el resultado en pantalla.

## Funcionalidad

El programa realiza las siguientes acciones:

1. Solicita el nombre del producto.
2. Solicita el precio del producto.
3. Solicita la cantidad del producto.
4. Valida que el precio y la cantidad sean valores numéricos válidos.
5. Calcula el costo total multiplicando el precio por la cantidad.
6. Muestra toda la información del producto en la consola.
